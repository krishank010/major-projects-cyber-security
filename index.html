05. Create a Log Monitoring and Alerting Script

Objective: Develop a Python or Bash script to monitor logs for suspicious activities and send alerts.

Python Tool:

  1. Use the os and re modules to read logs from /var/log/ or any other log directory.
  2. Search for patterns indicating:
    • Failed login attempts.
    • Unauthorized access.
    • System resource abuse (CPU, RAM spikes).
  3. Use smtplib to send email alerts for flagged entries.

Bash Tool:

  1. Use grep to filter specific patterns in logs (e.g., “Failed password” or “Invalid user”).
    • Example: grep "Failed password" /var/log/auth.log.
  2. Use mail or sendmail commands to notify administrators.

Additional Features (Optional):

  • Integrate with Slack or Telegram APIs for real-time alerts.
